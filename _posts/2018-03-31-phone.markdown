### 诺基亚5320
<a id="nokia5320"></a>

### iPhone 7
<a id="iphone7"></a>
工作一年左右买的手机，不买plus是因为屏幕太大，以及我对拍照没啥需求。对比手上的 iphone6，从外观上来主要是亮黑色很好看，机子看上去纯黑一体，以及摄像头边缘更加圆润了，总体来说还算满意。最喜欢 iphone7 home键上的线性马达，使用起来非常爽。这也是我开始第一款开始带壳的手机，没办法，亮黑色太爱刮花了。从这一代开始以后 iphone 我也不打算越狱了，现在的原生体验已经非常好了，不用瞎折腾了。

### BlackBerry Q10
<a id="BlackBerryQ10"></a>
通过我哥赞助买的手机，为了体验BB10，在 Q10 价格跌了一半后入手了。键盘的手感，手机的体验对我来说都是非常不错的，最大的缺点还是第三方软件太少，造成生活上不便。虽然系统兼容安卓软件，但是因为配置比较低，用安卓软件比较卡，而且安卓软件体验上也比较糟糕。之前一直当备用机用，最后慢慢被 priv 取代了备用机的地位。

### BlackBerry Priv
<a id="BlackBerryPriv"></a>
毕业后买的第一部手机，黑莓自家的第一部安卓手机，也可能是最后一部（后面的都是黑莓授权品牌的，说不准以后黑莓不授权了自己开发）。Priv 我当了一段时间主力机，808处理器是最大的坑点，整体手机做工也一般，按压背板有凹陷感，不过键盘信仰还是很足的。总体使用起来还算复合我的预期，不过我更喜欢 BB10 的系统。

### BlackBerry Key2
<a id="BlackBerryKey2"></a>
2018年6月12号首发立刻抢了一台银色。之前玩过同事的 Blackberry keyone，当时虽然很喜欢，但是没有种草，而且因为是TCL挂名研发生产的所以略有抵触。不过这次 key2 算是给我了点惊喜，方正的设计和银色的配色让我看到了 Q30 的身影（猛地一看也有点像Nokia Lumia 930），正面显示屏的更加一体外加speed key，配置和价钱还是那个尿性，配置低价钱贵（3999 = CPU660 + 6GB + 64GB），不过比外国的价格便宜，能看出TCL对中国市场的重视。

### 坚果 R1
<a id="jianguoR1"></a>
2018年5月15号手机发布的当天就抢了一台白色（算是弥补没有 T1 白色的遗憾吧），这次白色的设计再次维系了锤子"逢白必精"的品质。而这次作为旗舰，也终于第一次在短时间内就用上了安卓阵营的顶尖处理器。尽管这次在系统上没有很多新意，但是`系统原本的高品质` + `顶尖配置` + `浑然一体的白色外观设计`终于能让锤子在手上多当一段时间主力机了。以前配置这个最大的短板让手机基本上只能处在当备用机的水平。
